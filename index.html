ardeaXcel Web Server

ardeaXcel Web Server is small web server that created with VBA MS Excel. You can use it for publishing your excel file on the web or you can access it as bussiner layer for your web application, you can connect with curl.

Requirements:
1. Windows Operationg System
2. MS Excel
